Website Food Delight

project website sederhana untuk Food Delight, sebuah platform layanan makanan fiktif yang menampilkan berbagai hidangan makanan dan minuman. Website ini dirancang untuk responsif, mudah dinavigasi, dan menarik secara visual, dengan berbagai halaman yang memiliki menu navigasi untuk mengeksplorasi berbagai bagian seperti Beranda, Menu, dan Feedback.

Fitur
- Halaman Utama (Homepage): Menampilkan header selamat datang, menu navigasi, dan pengenalan tentang website.
- Halaman Menu:
    - Hidangan Utama: Menampilkan berbagai hidangan utama seperti Nasi Goreng, Beef Steak, Nasi Kuning, dan Spaghetti.
    - Minuman: Daftar minuman segar seperti Jus Mangga, Jus Apel, Soda, dan Lemon Tea.
- Feedback: Halaman untuk pengguna memberikan feedback tentang pengalaman mereka.

Teknologi yang Digunakan
- HTML5: untuk struktur dan konten halaman.
- CSS3: untuk styling halaman agar tampak bersih dan modern.
- Bootstrap 5: untuk layout grid responsif dan komponen seperti navbar, tombol, dan card.
- JavaScript: untuk menambah interaktivitas pada halaman, seperti validasi form atau peningkatan lainnya.
- Desain Responsif: web ini sepenuhnya responsif, sehingga dapat diakses baik di perangkat desktop maupun mobile.
